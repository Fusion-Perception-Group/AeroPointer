##四元数

| 向量       | 四元数                           | 说明                                   |
|------------|----------------------------------|----------------------------------------|
| (1, 0, 0)  | (0, 0, 0, 1)                     | 无需旋转，四元数为单位四元数。             |
| (0, 1, 0)  | (0, 0, sin(π/4), cos(π/4))       | 绕z轴逆时针旋转90度。                    |
| (0, 0, 1)  | (0, sin(π/4), 0, cos(π/4))       | 绕y轴逆时针旋转90度。                    |
